# 🎨 Air Canvas - 基于手势识别与 OpenCV 的空中画板

Python 版本：3.10.10

[English Version](./README_EN.md) | 中文文档

一个基于 **OpenCV** 和 **MediaPipe** 开发的无接触手势识别空中画板系统。用户可以通过摄像头使用简单的手势在空中绘制图形、选择画笔粗细与颜色，并支持透明背景画作保存与临摹图案实时相似度打分。

---

## ✨ 核心功能

* **✋ 智能手势识别**
  * **选择模式（食指 + 中指抬起）：** 用于悬停选中顶部 UI 菜单（字号、颜色、功能按钮）。
  * **绘画模式（仅食指抬起）：** 在推荐绘画区域进行流畅书写或擦除。
* **🎨 双层交互式 UI 菜单**
  * **第一层（粗细与保存）：** 提供 4px、8px、15px、25px 四种笔触粗细，以及 `SAVE` 保存按键。
  * **第二层（调色盘与功能）：** 支持红、橙、黄、绿、青、蓝、紫多种颜色的快速切换，并提供橡皮擦（ERASER）与一键清屏（CLEAR）。
* **💧 动态交互反馈**
  * 支持指尖悬停触发环形倒计时进度条。
  * 按钮选中时触发水波纹扩散动画。
* **🖼️ 临摹模式与实时打分算法**
  * 支持加载自定义背景底图，并将画板与底图融合展示。
  * 内置计算算法，基于精准度（Precision）、骨架覆盖率（Recall）及用墨量惩罚因子，实时计算临摹相似度得分。
* **💾 透明背景画作导出**
  * 支持自动提取非白色笔触，一键保存为 PNG 透明背景图片（不保存选择的背景图片）。

## 🛠️ 环境依赖与安装

请确保系统已安装 Python 3.8+ 环境，然后运行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

---

## 🚀 快速开始

将项目代码与所需中文字体文件（如 simhei.ttf 或 STCAIYUN.TTF）放在同一目录下。

运行主程序脚本：

```Bash
python ui_run_app.py
```

**先打开摄像头，保存人脸图片。进行LBPH检测，通过后可打开画布进行创作。**

打开画布后，将手置于摄像头前：

伸出食指和中指，悬停在顶部菜单项上约 0.3 秒完成选择。

仅伸出食指，在下方的【推荐绘画区域】内开始自由绘制。

```
Air Canvas/ 
├── __pycache__/           #Python 字节码缓存文件夹 
├── facedata/              # 人脸数据/样本存放目录 
├── haar/                  # OpenCV Haar 级联分类器模型目录 
├── model/                 # 训练好的模型文件存放目录 
├── Pic_example/           # 示例图片/临摹底图目录 
├── Pic_save/              # 绘画作品保存目录 
├── draw_ui_source.py      # UI 界面与绘图逻辑源码 
├── draw.py                # 核心绘图/手势处理逻辑 
├── icon.png               # 应用图标
├── mediapipe_draw.ui      # Qt Designer 生成的 UI 界面设计文件 
├── README_CH.md           # 中文项目说明文档 
├── README_EN.md           # 英文项目说明文档 
├── requirements.txt       # 项目依赖包列表 
├── STCAIYUN.TTF           # 中文字体文件（华文彩云） 
└── ui_run_app.py          # 主程序入口（启动 GUI 应用）
```
